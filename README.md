# Simple Task Manager

This is a simple Task Manager application built using Angular (Standalone Components).
The app supports basic CRUD operations and stores data using browser LocalStorage.

## Features
- Add new tasks
- Edit task title and description
- Change task status (To Do / In Progress / Done)
- Delete tasks
- Data persists using LocalStorage

## Tech Stack
- Angular 20
- TypeScript
- HTML & CSS
- LocalStorage (No backend)

## How to Run
1. npm install
2. ng serve
3. Open http://localhost:4200

## Notes
This project is intentionally kept simple and readable for assessment purposes.
